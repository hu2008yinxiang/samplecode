安装
Windows 下安装
在 http://www.phalconphp.com/en/download/windows 下载后安装


Linux 下安装
apt-get install libpcre3 libpcre3-dev php5-dev php5-mysql gcc
git clone --depth=1 git://github.com/phalcon/cphalcon.git
cd cphalcon/build
sudo ./install
